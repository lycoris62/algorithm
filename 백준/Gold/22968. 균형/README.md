### 문제 설명

<p>이진 탐색 트리의 한 종류인 AVL Tree는 "높이 균형 성질"이라는 성질을 이용해 트리의 균형을 맞춘다.</p>

<p>또한, 높이 균형 성질을 만족하는 이진 탐색 트리는 전부 AVL Tree이다.</p>

<p>위 그림에서, 왼쪽에 있는 트리는 모든 내부 정점의 왼쪽 부트리와 오른쪽 부트리의 높이가 동일하므로 AVL Tree이다.</p>
